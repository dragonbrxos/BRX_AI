# BRX AI - Professional Edition (Arch Linux) 🧬🐧

O **BRX AI** é um ecossistema de inteligência artificial nativo para Arch Linux, focado em leveza, transparência e colaboração. Esta versão **Professional Edition (v6.8)** traz um gerenciamento completo de ciclo de vida, um motor de pensamento atômico refinado e capacidades de programação drasticamente expandidas.

---

## 🚀 Gerenciamento de Ciclo de Vida

Agora você pode instalar, remover ou atualizar o BRX AI com um único comando profissional:

### Instalação Limpa
```bash
git clone https://github.com/dragonbrxos/BRX_AI.git && cd BRX_AI && chmod +x brx-manager.sh && sudo ./brx-manager.sh install
```

### Atualizar para a Versão Mais Recente
```bash
sudo brx-manager update
```

### Remover Completamente do Sistema
```bash
sudo brx-manager remove
```

### REMOVER REPOSITÓRIO BAIXADO 
```bash
rm -rf BRX_AI
git clone https://github.com/dragonbrxos/BRX_AI.git && cd BRX_AI && chmod +x brx-manager.sh && sudo ./brx-manager.sh install
```
